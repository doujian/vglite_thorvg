## Features Implemented
- [x] Win32 fullscreen window using CreateWindowEx with WS_POPUP style
- [x] Query primary monitor resolution at runtime using GetSystemMetrics(SM_CXSCREEN/SM_CYSCREEN)
- [x] Handle WM_CLOSE for window close button
- [x] Handle WM_KEYDOWN for ESC key to exit
- [x] Handle WM_PAINT for rendering
- [x] Initialize VGLite with SW backend (vg_lite_init, vg_lite_allocate)
- [x] Clear buffer to solid color (vg_lite_clear, vg_lite_finish)
- [x] Proper cleanup (vg_lite_free, vg_lite_close)
- [x] CMakeLists.txt builds the example successfully

## Known Limitations
- The example uses GDI for screen rendering (simple color fill) since SW backend 
  renders to a CPU buffer that requires manual blitting to screen
- Full VGLite rendering would require copying buffer->memory to screen via 
  GDI BitBlt or DirectX

## Build Configuration
- Backend: SW (VG_LITE_RENDER_BACKEND=1)
- Platform: Windows (VGLITE_PLATFORM_WINDOWS)
- Compiler: MSVC v145
- Architecture: x64
